# 将棋の縁台 🏠♟️

あたたかい将棋サロン風の将棋Webアプリです。AI対局を楽しめます。

## 特徴

- 🎨 木目調の温かいデザイン
- 📱 スマートフォン対応（レスポンシブ）
- 🤖 3段階のAI難易度（やさしい/ふつう/つよい）
- ♟️ 完全な将棋ルール実装（成り、打ち、王手、詰み判定）
- 📖 大きな文字とボタン（50代以上の方にも優しい）

## 技術スタック

- Next.js 14 (App Router) + TypeScript
- Tailwind CSS
- PWA対応準備済み

## セットアップ

```bash
npm install
npm run dev
```

http://localhost:3000 で開きます。

## AI難易度

| 難易度 | 説明 |
|--------|------|
| やさしい 🌱 | ランダムに合法手を選びます |
| ふつう ⚔️ | 駒の価値を考慮した評価関数で判断 |
| つよい 🔥 | ミニマックス+アルファベータ探索（深さ3） |

## ページ構成

- `/` - トップページ
- `/play` - AI対局画面
